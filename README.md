# Mini-Banking-System-OOP-
This is a small personal project created to apply and practice Object-Oriented Programming (OOP) principles using Java.

The project simulates a simple Mini Banking System where users can create an account, log in, and perform basic banking operations.

Features:

1.Sign Up and Login

2.Deposit money

3.Withdraw money

4.Transfer money to other accounts

5.Display account details such as holder name,account number and balance

Purpose
The main goal of this project is to strengthen my understanding of:

Classes and Objects

Encapsulation

File handling

Basic system design using OOP concepts

How to Run

Option 1: Using IntelliJ IDEA

Open the project in IntelliJ IDEA.

Make sure the accounts.txt and transactions.txt files are in the project root directory.

Open the Main class.

Click Run.

Option 2: Using Terminal (Command Line)

Navigate to the project directory.

Compile the project:
javac src/*.java
java src.Main
